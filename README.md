## 📋 项目简介

一个基于 Python + Selenium 的智能学习辅助工具，专为干部网络学习平台设计。该系统能够自动化完成课程学习流程，帮助用户高效管理学习进度。

## 🎯 主要功能

### 核心特性

- ✅ **自动登录系统** - 支持多账号轮换登录
- ✅ **智能课程识别** - 自动筛选未完成课程
- ✅ **进度自动上报** - 定时向服务器上报学习进度
- ✅ **实时进度监控** - 动态显示学习完成情况
- ✅ **完成状态验证** - 学习结束后确认课程状态

### 技术亮点

- 🛠 **浏览器自动化** - 基于Selenium的稳定操作
- 🛠 **异常处理机制** - 完善的错误捕获和重试逻辑
- 🛠 **灵活配置系统** - 支持参数自定义调整
- 🛠 **详细日志记录** - 完整的运行状态跟踪
